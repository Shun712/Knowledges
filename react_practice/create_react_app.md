# create_react_appとは？

簡単に`React`を学習する環境構築をしてくれるもの

# create_react_appに必要なもの

- Node.js: 10.16以上

- npm: 5.6以上

- Homebrewとnodebrewを使ってインストール

  - Homebrew: MacOSのパッケージ管理

  - nodebrew: node(Node.js)のバージョン管理ツール
